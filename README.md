# Getting started

## On a clean machine
With git:
```
xcode-select --install
git clone https://github.com/rensftw/.dotfiles
```
  
Without git:
```
mkdir .dotfiles
cd .dotfiles
curl -L https://api.github.com/repos/rensftw/.dotfiles/tarball | tar xz --strip=1
```
